# shadcn/ui

shadcn/ui es una coleccion de componentes para React (muy comun en Next.js) basada en Radix UI + Tailwind. La idea clave es que **no es una libreria que importas**, sino que **copias el codigo de los componentes a tu repo** para poder modificarlos.

## Por qué se usa

- Componentes accesibles (a11y) como base.
- Personalizacion real: el codigo vive en tu proyecto.
- Buen balance entre velocidad de desarrollo y control.

## Con qué suele ir de la mano

- Tailwind CSS
- Radix UI (primitivos)
