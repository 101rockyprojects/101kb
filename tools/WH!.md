# Tools

This folder is for tools, platforms, and libraries: what they are, when to use them, and practical notes (setup, pitfalls, snippets, links).

## What belongs here

- Products and services (e.g. queues, payment providers, CDNs).
- Frameworks, UI libraries, CLIs, deployment platforms.
- "How I use it" notes: defaults, conventions, quick commands, gotchas.

## Conventions

- Keep a `TL;DR` at the top if the tool is big.
- Prefer links to concepts instead of re-explaining fundamentals:
  - Example: link to [[Idempotencia]] from a note about message queues.
- Add `Related` links to adjacent tools (alternatives, complements).
