# Cloudinary

Cloudinary es una plataforma de gestion y entrega de assets (principalmente imagenes y video) para web y mobile. Te resuelve el "pipeline" completo:

- Upload (directo desde cliente o via backend).
- Almacenamiento y organizacion.
- Transformaciones on-the-fly (resize, crop, formatos, calidad).
- CDN y optimizacion para distintos dispositivos.

## Casos de uso tipicos

- Avatares y fotos de productos con multiples tamanos.
- Generacion de thumbnails y recortes automaticos.
- Optimizar performance (peso, formatos modernos, caching).

## Riesgos / Cosas a vigilar

- Seguridad: uploads firmados, restricciones de origen, sanitizar.
- Costes: transformaciones y ancho de banda pueden escalar rapido.
- Vendor lock-in: define bien convenciones de nombres y migracion.
