# CodeIgniter

CodeIgniter es un framework web de codigo abierto basado en PHP. Se caracteriza por ser ligero y rapido, con una curva de aprendizaje relativamente sencilla.

Suele usarse con el patron [[concepts/Modelo-Vista-Controlador (MVC)]] y ofrece routing, helpers/librerias, caching y un enfoque practico para construir apps web.

## Cuándo elegirlo

- Proyectos pequeños/medianos donde prima simplicidad y velocidad de entrega.
- Equipos que prefieren un framework con poca "magia" y una huella ligera.
