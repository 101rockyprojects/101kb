# Snippets

This folder is for copy-pastable pieces of code or config: small, reusable, and easy to search.

## What belongs here

- Minimal examples that you reuse often (functions, patterns, configs).
- "Known good" commands with context (what it does, when to run it).
- Templates: checklists, PR templates, small boilerplates.

## Conventions

- Make it runnable: include language fences (e.g. `bash`, `ts`, `go`).
- Include assumptions at the top (OS, versions, required env vars).
- If the snippet depends on a concept or tool, link to it:
  - `Related: [[tools/RabbitMQ]]`, `[[concepts/Event Driven]]`
