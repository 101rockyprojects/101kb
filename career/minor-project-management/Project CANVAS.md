# Project Canvas

Project Canvas es una herramienta visual para alinear a un equipo sobre un proyecto en una sola pagina: objetivos, entregables, riesgos, stakeholders, y el "plan" a alto nivel.

## Para qué sirve

- Mejorar la comunicacion del proyecto (especialmente al inicio).
- Detectar huecos temprano: riesgos, dependencias, alcance, supuestos.
- Servir como "foto" que se actualiza a medida que se aprende.

## Cuándo usarlo

- En proyectos con muchas partes interesadas.
- Cuando el equipo necesita un punto de referencia rapido (sin leer un documento largo).
