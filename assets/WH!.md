# Assets

This folder stores non-markdown attachments used by notes (mostly images).

## Images

Images live in `assets/img/` and are organized by where they are referenced from.

Example:
- A note under `career/courses/...` should reference images stored under:
  - `assets/img/career/courses/.../<image>.png`

This keeps the vault tidy and makes it easier to understand why an image exists.

## Conventions

- Prefer descriptive filenames when possible.
- Avoid duplicating the same image under multiple folders unless it is truly note-specific.
- If you reorganize images, update the links inside the notes (Obsidian and standard Markdown both work).
